# CAMELYON16 Data Set
## Overview
The goal of CAMELYON16 challenge is to evaluate new and existing algorithms for automated detection of metastases in hematoxylin and eosin (H&E) stained whole-slide images of lymph node sections. This task has a high clinical relevance but requires large amounts of reading time from pathologists. Therefore, a successful solution would hold great promise to reduce the workload of the pathologists while at the same time reduce the subjectivity in diagnosis.

For the complete description of the challenge and the data set please visit the [challenge](https://camelyon16.grand-challenge.org) website.

The data set is provided by the [Computational Pathology Group](http://www.diagnijmegen.nl/index.php/Digital_Pathology) of the Radboud University Medical Center in Nijmegen, The Netherlands.

## Data
### Images
The data in this challenge contains a total of 399 whole-slide images (WSIs) of sentinel lymph node from two independent data sets collected in Radboud University Medical Center (Nijmegen, The Netherlands), and the University Medical Center Utrecht (Utrecht, The Netherlands).

The slides are converted to generic [TIFF](https://www.awaresystems.be/imaging/tiff/bigtiff.html) (Tagged Image File Format) using an open-source file converter, part of the [ASAP](https://github.com/GeertLitjens/ASAP) package.

### Annotations
The shared XML files are compatible with the [ASAP](https://github.com/computationalpathologygroup/ASAP) software. You may download this software and visualize the annotations overlaid on the whole slide image.

The provided XML files may have two groups of annotations ("Tumor", or "Exclusion") which can be accessed from the "PartOfGroup" attribute of the Annotation node in the XML file. Annotations belonging to group "Tumor" represent tumor areas and annotations within group "Exclusion" are non-tumor areas which have been cut-out from the original annotations in the "Tumor" group.

### Masks
A mask file is shared for each WSI in the data set. The maks files have value 1 for normal tissue, and 2 for tumor in the corresponding WSI. Normal tissue mask is not provided for the not exhaustively anntotated WSIs. The normal tissue regions were automatically detected as in [10.7717/peerj.8242](https://doi.org/https://doi.org/10.7717/peerj.8242), therefore there can be errors in the masks. The tumor regions were generated by converting the annotation polygons to pixel masks.

CAMELYON16 数据集
概述

CAMELYON16 挑战赛的目标是评估用于自动检测淋巴结切片中转移灶的新算法与已有算法。数据基于苏木精-伊红（H&E）染色的淋巴结全视野切片（Whole-Slide Images, WSIs）。
该任务具有很高的临床相关性，但传统人工阅片需要病理医生投入大量时间。因此，一个成功的自动化解决方案有望在降低病理医生工作负担的同时，减少诊断过程中的主观性。

关于挑战赛及数据集的完整说明，请访问官方挑战赛网站
。

本数据集由荷兰奈梅亨 Radboud 大学医学中心的计算病理学研究组（Computational Pathology Group）提供。

数据说明
图像

本挑战赛数据共包含 399 张哨兵淋巴结全切片图像（WSI），来源于两个相互独立的数据集，分别采集自：

Radboud 大学医学中心（奈梅亨，荷兰）

乌得勒支大学医学中心（乌得勒支，荷兰）

所有切片均通过开源文件转换工具（ASAP 软件包的一部分）转换为通用的 TIFF（Tagged Image File Format） 格式。

标注（Annotations）

共享的 XML 标注文件与 ASAP 软件兼容。你可以下载该软件，在全切片图像上可视化对应的标注信息。

XML 文件中的标注可能属于两类分组之一：

“Tumor”：肿瘤区域

“Exclusion”：从 “Tumor” 原始标注中剔除的非肿瘤区域

标注所属分组可通过 XML 文件中 Annotation 节点的 PartOfGroup 属性进行区分。

掩膜（Masks）

数据集中为每一张 WSI 提供了一个对应的掩膜文件：

像素值 1 表示正常组织

像素值 2 表示肿瘤区域

对于未进行穷尽标注的 WSI，不提供正常组织的掩膜。正常组织区域是通过自动方法检测得到的（参考文献 DOI: 10.7717/peerj.8242），因此掩膜中可能存在一定误差。
肿瘤区域掩膜则由标注多边形转换为像素级掩膜生成。
